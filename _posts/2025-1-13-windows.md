## 2.1 scoop包管理器
- 安装方法：[https://scoop.sh]([https://](https://scoop.sh/))
- 使用方法：
```sh
# 更新scoop
scoop update

# 更新所有软件
scoop update *

# 安装软件
scoop install packagename

# 卸载软件
scoop uninstall packagename
```
- 加速下载方法：
① 参考[2.2 网络配置](#2.2)
② 使用scoop-cn :
    ```sh
    scoop bucket add scoop-cn https://ghproxy.cc/https://github.com/duzyn/scoop-cn
    ```
